# Edit Entry Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an inline edit icon to every record row so users can correct in/out times; editing a submitted record reverts it to pending for re-submission.

**Architecture:** All changes are in `index.html`. A module-level `editingDate` variable tracks which row (if any) is in edit mode. `renderRecords()` is updated to tag each row's times span and actions span with IDs, and three new functions (`editRecord`, `saveRecord`, `cancelEdit`) manage the inline edit lifecycle. `editRecord` uses DOM methods (not string templates) to create inputs and wire event handlers.

**Tech Stack:** Vanilla JS, HTML, CSS — no build step, no test framework. Verification is done by opening `index.html` directly in a browser.

- [ ] **Step 2: Add `editRecord`, `saveRecord`, and `cancelEdit` after the `deleteRecord` function**

```js
function editRecord(date) {
  if (editingDate) cancelEdit(editingDate);
  editingDate = date;
  const r = getAllRecords()[date];

  const timesEl   = document.getElementById('times-'   + date);
  const actionsEl = document.getElementById('actions-' + date);

  const inInput  = document.createElement('input');
  inInput.type   = 'time';
  inInput.id     = 'edit-in-' + date;
  inInput.value  = r.in  || '';

  const outInput  = document.createElement('input');
  outInput.type   = 'time';
  outInput.id     = 'edit-out-' + date;
  outInput.value  = r.out || '';

  timesEl.textContent = '';
  timesEl.append('\uD83D\uDFE2 ', inInput, '\u00a0 \uD83D\uDD34 ', outInput);

  const saveBtn = document.createElement('span');
  saveBtn.className   = 'save';
  saveBtn.textContent = '\u2713';
  saveBtn.onclick     = () => saveRecord(date);

  const cancelBtn = document.createElement('span');
  cancelBtn.className   = 'cancel';
  cancelBtn.textContent = '\u2717';
  cancelBtn.onclick     = () => cancelEdit(date);

  actionsEl.textContent = '';
  actionsEl.append(saveBtn, cancelBtn);
}

function saveRecord(date) {
  const recs       = getAllRecords();
  recs[date].in    = document.getElementById('edit-in-'  + date).value;
  recs[date].out   = document.getElementById('edit-out-' + date).value;
  recs[date].submitted = false;
  editingDate      = null;
  saveAllRecords(recs);
  renderRecords();
}

function cancelEdit(date) {
  editingDate = null;
  renderRecords();
}
```

Unicode reference: `\uD83D\uDFE2` = 🟢, `\uD83D\uDD34` = 🔴, `\u2713` = ✓, `\u2717` = ✗, `\u00a0` = non-breaking space.
